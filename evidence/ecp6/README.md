# ECP-6 compact evidence

This directory contains a reviewable snapshot of the ECP-6 confirmatory result. Large checkpoints, isolated process matrices and 163,840 episode records are deliberately not stored in Git.

- `manifest.json` records the frozen hashes, seed outcomes and integrity checks.
- `baselines.json` records the representation baselines generated by the run.
- `report.md` is the compact generated experiment report.

The original local run directory was `runs/20260718T070018Z-ecp6-experiment`. Reproduce it with the frozen configuration:

```bash
ecp6 --config config/ecp6.yaml experiment --unseal-test
ecp6 --config config/ecp6.yaml analyze runs/<generated-run> --permutations 100
```

The documented SHA-256 values allow the frozen config and deterministic split to be checked independently.
